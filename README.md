# OSCE Oracle

Free revision tool for UK medical students preparing for OSCEs. Work through examination steps, see the questions an examiner might ask, and practice with full mock stations.

**Live:** https://amacrutherford.github.io/osce

## Features

- Step-by-step guides for 30+ OSCE examinations across orthopaedics, neurology, gynaecology, paediatrics, psychiatry, and more
- Examiner questions at each step — rationale questions, common pathologies, and anatomical findings
- Full mock OSCE stations with candidate brief, actor script, mark scheme, and viva questions — based on past Y5 papers
- Study and exam modes, high-yield summaries, progress tracking saved to browser localStorage
- No account needed, no backend, completely free

## Run locally

```bash
cd osce
npm install
npm run dev
```

**Stack:** React 19, TypeScript, Tailwind CSS, Vite. All content is static TypeScript — no database or API.
